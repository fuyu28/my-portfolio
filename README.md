# Portfolio (Astro + React + Tailwind + TypeScript)

## Setup

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```

## Notes

- Edit content in `src/pages/index.astro`.
- Styling lives in `src/styles/global.css`.
- React component example in `src/components/HighlightGrid.tsx`.
